# 駅ガチャ (Station Gacha)

「今日はどこに行こう？」と迷ったときに。
ボタンをポチッと押すだけで、東京23区内や市町村の駅をランダムに選んでくれるWebアプリです。

## 特徴
* **ランダム抽選**: ボタンひとつで駅を「エイヤ！」で決定。
* **エリア絞り込み**: 東京23区、市町村、その他（未特定）の中から行きたいエリアをチェックボックスで選べます。
* **簡単アクセス**: 表示された駅名をクリックすると、Google Mapsで周辺環境をすぐチェック。
* **自分専用リスト**: 行ったことのある駅をチェックすれば、次は「まだ知らない駅」だけをガチャできます。
* **レスポンシブ対応**: PCでもスマホでも見やすいデザイン。

## 技術スタック
* HTML5 / CSS3 / JavaScript (Vanilla)
* データ: 鉄道駅オープンデータを利用

## 使い方
1. ページ内のチェックボックスから、行きたいエリア（区・市町村）を選択します。
2. 「駅ガチャを回す」ボタンを押して、今日の行先を決めます。
3. 行ったことがある駅はチェックを入れて、「行ったことのない駅のみ表示」にチェックを入れると未訪問の駅だけが抽選されます。
