# 駅ガチャ (Station Gacha)

「今日はどこに行こう？」と迷ったときに。
ボタンをポチッと押すだけで、東京23区内の駅をランダムに選んでくれるWebアプリです。

## 特徴
* **ランダム抽選**: ボタンひとつで23区内の駅を「エイヤ！」で決定。
* **簡単アクセス**: 表示された駅名をクリックすると、Google Mapsで周辺環境をすぐチェック。
* **自分専用リスト**: 行ったことのある駅をチェックすれば、次は「まだ知らない駅」だけをガチャできます。
* **軽量・高速**: フレームワークを使わないシンプルな構成で、サクサク動きます。

## 使い方
1. 公開されたURLにアクセス。
2. 「駅ガチャを回す」ボタンを押して、今日の行先を決める。
3. 行ったことがある駅はチェックを入れて、「行ったことのない駅のみ表示」にチェックを入れると未訪問の駅だけが抽選されます。

## 技術スタック
* HTML5 / CSS3 / JavaScript (Vanilla)
* データ: 鉄道駅オープンデータを利用
